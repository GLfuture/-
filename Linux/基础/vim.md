# VIM:一个编辑器

* 命令模式					一打开就是

* 输入模式            		 

* 底行命令模式

  命令模式------>输入模式

  a	i	o

  输入模式-------->命令模式

  esc

  命令模式-------->底行命令模式

  :

  底行命令模式：

  :w保存

  :q退出

  :x保存并退出

  :set nu设置显示行号

  :set tabstop=n设置tab键的距离为n

  命令模式：

  dd剪切

  ndd剪切n行

  yy拷贝

  nyy拷贝n行

  yw拷贝一个单词(以空格划分)

  nyw拷贝n个单词

  p粘贴

  /(或者?)查找内容(查找文本中匹配内容)
